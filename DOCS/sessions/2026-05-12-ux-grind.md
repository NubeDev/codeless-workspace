  `WorktreeManager::new` from serve.rs. UI: `WorktreeGcButton` on
  the dashboard header opens a confirm modal that loads a dry-run
  preview first; the "Remove N" action only enables once preview

  list. Test coverage: 4 tokio tests pin the no-root error,
  dry-run no-op, real-run removal, and age-filter exclusion.
